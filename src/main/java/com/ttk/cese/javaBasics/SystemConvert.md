### 十进制转二进制
> 13转2进制, 用13一直除以2, 直到商=0为止, 余数从下至上就是2进制的值

| 除数 | 被除数  |  商  |  余数  |
| :--- | :---: |  :---:  |  :---:  |
|  13  | 2 |  6  |  1  |
|  6  | 2 |  3  |  0  |
|  3  | 2 |  1  |  1  |
|  1  | 2 |  0  |  1  |

### 二进制转十进制
> 1001101转10进制

| 数 | 权值  | 值  |
| :--- | :---: | :---: |
|  1  | 6 | 1 * 2^6 |
|  0  | 5 | 0 * 2^5 |
|  0  | 4 | 0 * 2^4 |
|  1  | 3 | 1 * 2^3 |
|  1  | 2 | 1 * 2^2 |
|  0  | 1 | 0 * 2^1 |
|  1  | 0 | 1 * 2^0 |
